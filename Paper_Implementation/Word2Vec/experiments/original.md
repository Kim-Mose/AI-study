# 원본 결과 재현 실험

## 목표
Word2Vec(CBOW, Skip-gram)을 PyTorch로 구현하여 단어 임베딩을 학습한다.

## 데이터셋 후보
- **Wikipedia Dump** - 대용량 영어/한국어
- **PennTreeBank** - 작은 규모 영어 코퍼스
- **NSMC** (네이버 영화 리뷰) - 한국어
- **간단한 예시 코퍼스** - 학습 동작 확인용

## 실험 설정

### 모델 비교
| 항목 | CBOW | Skip-gram |
| --- | --- | --- |
| 입력 | 주변 단어 | 중심 단어 |
| 출력 | 중심 단어 | 주변 단어 |
| 학습 속도 | 빠름 | 느림 |

### 하이퍼파라미터
| 항목 | 값 |
| --- | --- |
| Embedding Dim | 100 |
| Window Size | 5 |
| Optimizer | Adam |
| Epochs | 100 |
