# 논문 결과 재현 실험

## 목표
AlexNet의 핵심 구조를 PyTorch로 구현하여 분류 성능을 확인한다.<br>
원본은 ImageNet (1000 클래스, 120만 장)이지만 자원 한계로 **CIFAR-10**으로 진행.

## 실험 설정

### 데이터셋
- **CIFAR-10** (32×32 → 224×224 리사이즈)
- 10개 클래스
- 학습: 50,000장, 테스트: 10,000장

### 모델
원본 AlexNet 구조 (출력층만 1000 → 10 변경)

### 하이퍼파라미터
| 항목 | 값 |
| --- | --- |
| Optimizer | Adam |
| Learning Rate | 0.001 |
| Batch Size | 128 |
| Epochs | 20 |

## 분석
- ImageNet 데이터가 아니라 직접 비교는 어려움
- CIFAR-10 같은 작은 이미지에선 AlexNet이 과한 모델일 수 있음
- 작은 데이터셋에서는 ResNet, VGG가 더 효율적

## 결론
AlexNet 구조의 핵심 컴포넌트(ReLU, Dropout, 큰 필터)가 잘 동작함을 확인.<br>
다만 작은 이미지(32×32)에는 11×11 같은 큰 필터가 효율적이지 않음.
