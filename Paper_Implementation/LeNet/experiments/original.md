# 논문 결과 재현 실험

## 목표
원본 LeNet-5 논문의 MNIST 결과를 PyTorch로 재현한다.

## 실험 설정

### 데이터셋
- **MNIST** (28×28 흑백 손글씨 숫자)
- 학습: 60,000장
- 테스트: 10,000장

### 모델 (논문 원본 구조)
| 층 | 입력 | 출력 | 비고 |
| --- | --- | --- | --- |
| Conv1 | 28×28×1 | 28×28×6 | 5×5, padding=2 |
| AvgPool | 28×28×6 | 14×14×6 | 2×2, stride=2 |
| Conv2 | 14×14×6 | 10×10×16 | 5×5 |
| AvgPool | 10×10×16 | 5×5×16 | 2×2, stride=2 |
| FC1 | 400 | 120 | Sigmoid |
| FC2 | 120 | 84 | Sigmoid |
| FC3 | 84 | 10 | - |

활성화 함수: **Sigmoid** (논문 원본대로)

### 하이퍼파라미터
| 항목 | 값 |
| --- | --- |
| Optimizer | Adam |
| Learning Rate | 0.001 |
| Batch Size | 64 |
| Epochs | 10 |
| Loss | CrossEntropyLoss |

### 환경
- PyTorch 2.x
- Mac M1 / M2 (MPS) 또는 CUDA

- 원본 논문의 옵티마이저는 SGD였음 (본 실험은 Adam 사용 → 더 빠른 수렴)
- 원본은 RBF 출력층 + 특수 손실함수 → 본 실험은 Softmax + CE

## 결론
PyTorch로 LeNet-5를 재현한 결과 **논문과 유사한 성능을 달성**했음을 확인했다.<br>
간단한 모델이지만 MNIST 같은 단순 데이터셋에서는 충분히 잘 동작함을 알 수 있다.
