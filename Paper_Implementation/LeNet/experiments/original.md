# 논문 결과 재현 실험

## 목표
원본 LeNet-5 논문의 MNIST 결과를 PyTorch로 재현한다.

## 실험 설정

### 데이터셋
- **MNIST** (28×28 흑백 손글씨 숫자)
- 학습: 60,000장
- 테스트: 10,000장

### 모델 (논문 원본 구조)
| 층 | 입력 | 출력 | 비고 |
| --- | --- | --- | --- |
| Conv1 | 28×28×1 | 28×28×6 | 5×5, padding=2 |
| AvgPool | 28×28×6 | 14×14×6 | 2×2, stride=2 |
| Conv2 | 14×14×6 | 10×10×16 | 5×5 |
| AvgPool | 10×10×16 | 5×5×16 | 2×2, stride=2 |
| FC1 | 400 | 120 | Sigmoid |
| FC2 | 120 | 84 | Sigmoid |
| FC3 | 84 | 10 | - |

활성화 함수: **Sigmoid** (논문 원본대로)

### 하이퍼파라미터
| 항목 | 값 |
| --- | --- |
| Optimizer | Adam |
| Learning Rate | 0.001 |
| Batch Size | 64 |
| Epochs | 10 |
| Loss | CrossEntropyLoss |

### 환경
- PyTorch 2.8
- Mac mini (MPS)

## 분석

### 논문 결과와의 비교
- 논문: 0.95% 에러 = 99.05% 정확도
- 본 구현: 1.51% 에러 = 98.49% 정확도

### 약간의 차이가 발생한 원인
- 원본 논문의 옵티마이저는 SGD with momentum이었음 (본 실험은 Adam)
- 원본은 RBF 출력층 + 특수 손실함수 → 본 실험은 Softmax + CrossEntropy
- 학습 epoch 수 (논문은 더 길게 학습)
- 데이터 증강 미적용

### 학습 안정성 관찰
- Sigmoid 활성화 함수임에도 첫 epoch부터 92% 달성
- 에폭이 진행될수록 안정적으로 수렴
- Train과 Test의 격차가 작음 → 과적합 없음

## 결론
PyTorch로 LeNet-5를 재현한 결과 **테스트 정확도 98.49%**를 달성했다.<br>
논문 성능(99.05%)과 0.5%p 차이로, 추가 epoch이나 더 정밀한 하이퍼파라미터 튜닝으로 보완 가능한 수준이다.<br>
간단한 모델이지만 MNIST에서 매우 좋은 성능을 보임을 확인.
