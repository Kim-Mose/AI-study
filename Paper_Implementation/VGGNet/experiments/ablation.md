## 실험 2: 3×3 vs 5×5 vs 7×7

### 가설
3×3 두 개 = 5×5 한 개와 같은 receptive field지만 3×3 두 개가 더 좋을 것이다.

### 설정
같은 receptive field를 가진 모델들 비교:
- A: 3×3 두 개
- B: 5×5 한 개

### 결과
| 필터 구성 | 파라미터 | Test Acc |
| --- | --- | --- |
| 3×3 × 2 |  |  |
