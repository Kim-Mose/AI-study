# 논문 결과 재현 실험

## 목표
VGG16 구조를 PyTorch로 구현하여 분류 성능을 확인한다.<br>
원본은 ImageNet이지만 자원 제약으로 **CIFAR-10** 사용.

## 실험 설정

### 데이터셋
- CIFAR-10 (224×224 리사이즈)

### 모델
- VGG16 (출력층 1000 → 10 변경)

### 하이퍼파라미터
| 항목 | 값 |
| --- | --- |
| Optimizer | Adam |
| Learning Rate | 0.001 |
| Batch Size | 64 |
| Epochs | 20 |

## 분석
- VGG16은 파라미터가 많아 작은 데이터셋에서 과적합 위험 있음
- 깊은 모델이라 학습 시간이 오래 걸림

## 결론
3×3 필터를 깊게 쌓는 단순한 구조가 효과적임을 확인.
