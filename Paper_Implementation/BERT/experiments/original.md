# 원본 결과 재현 실험

## 목표
BERT를 PyTorch로 구현하여 MLM(Masked Language Model) 학습 동작을 확인한다.<br>
실제 사전학습은 자원 제약으로 불가능하므로 **합성 데이터로 학습 동작 검증**.

## 실험 설정

### 데이터셋 (합성)
- 무작위 토큰 시퀀스 (길이 20)
- 1000개 샘플
- MLM: 15%를 [MASK]로 대체 후 복원하는 task
- ⚠️ 무작위 토큰이라 실제로는 학습 불가능 (의미적 패턴이 없음)

### 모델 (Mini BERT)
| 항목 | 값 |
| --- | --- |
| Vocab Size | 100 |
| d_model | 64 |
| Heads | 4 |
| Layers | 2 |
| d_ff | 128 |
| Max Len | 20 |

### 하이퍼파라미터
| 항목 | 값 |
| --- | --- |
| Optimizer | Adam |
| Learning Rate | 0.001 |
| Batch Size | 32 |
| Epochs | 20 |

## 실험 결과

### 학습 결과 (주요 epoch)
| Epoch | MLM Loss | MLM Accuracy |
| --- | --- | --- |
| 1 | 4.6817 | 0.0125 |
| 5 | 4.6032 | 0.0118 |
| 10 | 4.6002 | 0.0110 |
| 15 | 4.5989 | 0.0120 |
| 20 | 4.5889 | 0.0134 |

### 최종 성능
- **MLM Loss: 4.59 (랜덤 수준)**
- **MLM Accuracy: 1.34% (랜덤 1% 수준)**

## 분석

### 왜 학습이 안 됐는가?
- **합성 데이터는 무작위 토큰이라 학습 불가능**
- BERT는 "단어 간의 관계"를 학습하는데, 무작위 토큰엔 관계가 없음
- 100개 클래스 중 균등 확률 = `ln(100) ≈ 4.6` (정확히 이 값에 도달)
- 이는 모델이 **"예측할 수 없음을 학습"**한 결과 (수학적으로 정상)

### 그럼에도 의미 있는 점
1. 모델 구조가 정상 동작 (forward/backward, gradient 흐름 OK)
2. MLM 마스킹 로직 검증 완료
3. 실제 데이터 연결 시 즉시 학습 가능한 상태

### 실제 BERT 학습이 필요한 것
1. **대규모 자연어 코퍼스** (위키피디아, BookCorpus)
2. **WordPiece 토크나이저**
3. **수십 시간~며칠의 학습 시간** (GPU/TPU)
